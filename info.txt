Структура проекта

src/api - код для апи запросов
        --auth.js - содержит в себе все запросы для регистрации
        --axios.js - содержит базовые настройки для переиспользования (прим. базовый url для post запросов)

src/assets - подключаемые файлы (стили, каринки)
src/pages - страницы
src/components - компоненты

src/store - файлы работы со стором vuex
        --modules/auth.js - модуль для работы с авторизацией, содержит в себе мутациии(статусы регистрации) и экшены, экшен вызывается в компоненте -> делаем запрос который описан в файле src/api -> экшен вызывает мутацию и отдает ответ -> мутация вносит измения в state
        --index.js - хранит в себе список модулей, создает глобальный store

src/helpers - методы для работы с localStorage
